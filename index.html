今日やったこと
$ ssh seto@192.168.11.63 # サーバーにSSH接続
$ sudo -u postgres psql # PostgreSQLに接続
# \l # データベース一覧を表示
# CREATE DATABASE mydb; # データベースを作成
# \c mydb # mydbに接続（connect）
# CREATE TABLE users (id SERIAL PRIMARY KEY, name TEXT, email TEXT); # テーブル作成
# \dt # テーブル一覧を表示
ポイント： psqlの中のコマンドは \（バックスラッシュ）始まり。SQLは ;（セミコロン）で終わる。
キーワード・用語
コマンド／用語意味
sudo管理者権限で実行する
sudo -u postgrespostgresユーザーとして実行する
psqlPostgreSQLに接続するコマンド
\lデータベース一覧を表示（list）
\c [DB名]指定したデータベースに接続（connect）
\dtテーブル一覧を表示
\qpsqlを終了（quit）
CREATE DATABASEデータベースを作成するSQL
CREATE TABLEテーブルを作成するSQL
SERIAL自動で連番を振る型（1,2,3...）
PRIMARY KEYレコードを一意に識別するキー
TEXT文字列を格納するデータ型
